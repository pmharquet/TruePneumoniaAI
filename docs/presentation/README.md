# Présentation — TruePneumoniaAI

Diaporama de soutenance (grand public / jury non technique), en français.

## Présenter

Ouvrez simplement `index.html` dans un navigateur (double-clic). Aucune installation,
aucune connexion internet requise — tout est autonome.

| Action | Touche |
|---|---|
| Diapo suivante | `→`, `Espace` |
| Diapo précédente | `←` |
| Première / dernière | `Début` / `Fin` |

Sur écran tactile : balayer gauche/droite.

## Exporter en PDF

Dans le navigateur : `Ctrl+P` → destination « Enregistrer au format PDF » →
orientation **paysage**, marges **aucune**, cocher « graphiques d'arrière-plan ».
Chaque diapo devient une page.

## Contenu (16 diapos)

Titre → Problème → Objectif → Démarche → Données → Analyse (anti-fuite) →
Transfert d'apprentissage → Préparation des images → Entraînement → Seuil clinique →
Résultats (modèle principal) → Étage 2 (bactérien/viral) → Explicabilité (Grad-CAM) →
Produit fini → Limites → Conclusion.

Les images sont dans `assets/` (exemples de radios + aperçu des augmentations).
